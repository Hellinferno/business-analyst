# Business Analyst Assistant

AI-powered tools to accelerate Business Analysis work. See [ba-assistant/README.md](ba-assistant/README.md) for full setup and usage instructions.

## What's Inside

The [`ba-assistant/`](ba-assistant/) directory contains the full application:

- **Backend** — FastAPI + Python 3.11 REST API with Google Gemini AI integration
- **Frontend** — Next.js 14 React application with Tailwind CSS

## Quick Start

```bash
cd ba-assistant
docker compose up --build
```

Then open <http://localhost:3000>.
